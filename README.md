## 🚀 Installation

```bash
git clone https://github.com/Devanshuthakral/AI_CHATBOT

cd AI-Chatbot

pip install -r requirements.txt

streamlit run app.py
```

---

## 🔑 Environment Variables

Create a `.env` file:

```env
OPENAI_API_KEY=your_key
GOOGLE_API_KEY=your_key
ANTHROPIC_API_KEY=your_key
HF_TOKEN=your_key
```
